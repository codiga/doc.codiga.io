## All Languages

### Print an object

Use the function `printObject` to inspect the object in the console.

Example of usage

```javascript
function visit(node, filename, code) {
  printObject(node);
}
```

## Python

### useModule

Return `true` if a module is used.

```javascript
function useModule(imports, moduleName);
```

Example

```javascript
if (!useModule(node.context.imports, "boto3")) {
  return;
}
```

### findVariableFromFunctionCall

```
function findVariableFromFunctionCall(assignments, module, functionName)
```

Find a variable that has been assigned from a function call.

```javascript
const clientVariable = findVariableFromFunctionCall(
  node.context.assignments,
  "boto3",
  "client"
);
```

### useImportFrom

```javascript
function useImportFrom(imports, package, element);
```

Example

```javascript
useImportFrom(node.context.imports, "requests", "get");
```
