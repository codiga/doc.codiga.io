## Playground

Codiga provides a [playground](https://app.codiga.io/hub/playground) to experiment our
static analyzer and write your own rules. Use the playground to test your own rules,
experiment by starting from other people rules and let your imagination write the best
code analysis rules!

## Explore the Hub

The [Codiga Hub](https://app.codiga.io/hub/rulesets) hosts hundreds of custom
analysis rules you can reuse. Browse rules on the hub, import them in the
playground and create new analysis rules for your team!
